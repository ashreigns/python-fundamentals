# Python Fundamentals & Logic Exercises

Welcome to my personal Python development repository! This project is a comprehensive collection of core Python scripts, structured algorithms, logic exercises, and interactive mini-games designed to master the fundamentals of the language.

## 🚀 What's Inside?

The repository is organized into weekly modules, each focusing on key programming concepts:

* **Week 1 - Variables, Data Types, Input - Output:** Fundamental building blocks, basic arithmetic, and user interactions.
* **Week 2 - Conditional Statements (if - elif - else):** Control flow logic, validation systems, and decision-making structures.
* **Week 3 - Loops (for - while):** Iteration mechanics, prime number analysis, and series generation.
* **Week 4 - Lists and Tuples:** Data structure manipulation, mathematical statistics, and manual sorting algorithms (Bubble Sort).
* **Week 5 - Functions:** Modular code design, EBOB/EKOK (GCD/LCM) calculations, and recursion analysis.
* **Week 6 - String Operations:** Text processing, palindrome checks, and cryptography basics (Caesar Cipher).
* **Week 7 - Dictionaries:** Key-value data management, frequency mapping, and structural menu systems.
* **Week 8 - Mini Projects:** Full-fledged CLI games and interactive tools combining all core concepts.

## 🛠️ Tech Stack & Environment
